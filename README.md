# 🚀 Job Analyzer AI

An intelligent AI-powered system that analyzes job descriptions and predicts suitable job roles using Machine Learning and NLP.   LIVE: https://huggingface.co/spaces/tansin99/Job_Analyzer 

---

## 📌 Project Overview

**Job Analyzer AI** is designed to:

* Extract insights from job descriptions
* Predict job titles using trained ML models
* Help users understand job requirements efficiently

This project combines **Natural Language Processing (NLP)** with **Machine Learning** to deliver accurate predictions.

## 🧠 Features

* 🔍 Job description analysis
* 🤖 ML-based job title prediction
